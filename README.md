### 1. Correction Parameters Study

–°—Ä–∞–≤–Ω–µ–Ω–∏–µ 10 —Ä–µ–∂–∏–º–æ–≤ correction parameters –Ω–∞ —ç—Ç–∞–ø–µ data finetuning: None, Per-Step Bias, Per-Step Gate, Time Bias, Time Gate, RHS Scale, Output Bias –∏ –∏—Ö –∫–æ–º–±–∏–Ω–∞—Ü–∏–∏. 

–ë–∞–∑–æ–≤–∞—è —Å—Ö–µ–º–∞ –∏–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞–Ω–∏—è –Ω–∞ –∫–∞–∂–¥–æ–º —à–∞–≥–µ `i`:

–ü—Ä–∏ `g_i ‚Üí ‚àí‚àû`: `Œ∏_i ‚âà Œ∏_base` (–≥–µ–π—Ç –∑–∞–∫—Ä—ã—Ç). –ü—Ä–∏ `g_i ‚Üí +‚àû`: `Œ∏_i ‚âà œÉ(Œº_i)` (–ø–æ–ª–Ω–∞—è –∑–∞–º–µ–Ω–∞ –Ω–∞ –æ–±—É—á–∞–µ–º—É—é —Ü–µ–ª—å).

### –ö–æ—Ä—Ä–µ–∫—Ü–∏–∏ –≤—ã—Ö–æ–¥–∞ (–∏–∑–º–µ–Ω—è—é—Ç —Ä–µ—à–µ–Ω–∏–µ y, –Ω–µ —Ç—Ä–æ–≥–∞—è Œ∏)

**6. RHS Scale** ‚Äî –º—É–ª—å—Ç–∏–ø–ª–∏–∫–∞—Ç–∏–≤–Ω–æ–µ –º–∞—Å—à—Ç–∞–±–∏—Ä–æ–≤–∞–Ω–∏–µ –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ –Ω–∞ –∫–∞–∂–¥–æ–º —à–∞–≥–µ (`n_steps` –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤: `Œµ_i`):
